# Stratum
Mentalogue Stratum - Version Manager for Mentalogue.

See `RELEASE_NOTE.md` for the current release and `RELEASE_NOTES.md` for full release history.

## Usage

```powershell
stratum install 0.2.0
stratum use 0.2.0
stratum pin 0.2.0
stratum list
stratum which
```

Tip: you can install via `mntpack sync MentalogueLang/Stratum -g -r auto`.
